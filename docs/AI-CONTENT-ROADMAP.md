## Topic generation rules for the article-writer

When the daily-article-prompt.md prompt runs, it should:

1. Read this roadmap as the **starting palette**, not a fixed list.
2. Check current AI news from the last 30 days — if something big dropped (new Claude model, new MCP server, new agent platform), prioritize that.
3. Avoid duplicating slugs that already exist in `posts.json`.
4. For each article, pick a topic that ties an angle from this roadmap to specific current news, named tools, or a specific demand signal you found.
5. If a topic from this roadmap feels stale, replace it with a fresher angle. The roadmap is a guide, not handcuffs.
